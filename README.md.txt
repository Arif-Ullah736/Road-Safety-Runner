# Road Safety Runner

Road Safety Runner is a 2D Unity educational game created as a semester project.

## Objective
The objective of the game is to teach basic road safety awareness in an interactive way. The player must avoid moving cars, collect all safety signs, and reach school safely before the timer ends.

## Controls
- Arrow Keys / WASD: Move player

## Features
- Main Menu
- Player movement
- Moving cars
- Lives system
- Timer system
- Safety sign collection
- Warning message
- Win screen
- Game Over screen
- Background music and sound effects
- Button click sounds

## Win Condition
The player wins after collecting all 5 safety signs and reaching the school.

## Lose Condition
The player loses if all lives are finished or the timer reaches zero.
